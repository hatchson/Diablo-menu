## 🔥 Features

Full feature list → [View complete features here](https://hatchson.github.io/DiabloMenu/features.txt)

*(Once the link is live, this section will show the latest features directly. For now, here's a preview/example structure based on typical GT menus — update this when your txt file is online!)*

- **Gun & Pointer System**
  - Dynamic wavy energy beam (animated plasma effect)
  - Lock-on targeting with VRRig support
  - Color-changing pointer based on state/trigger

- **Visual Customization**
  - Real-time color cycling (buttons, background, lines)
  - Custom shaders (GUI/Text Shader for clean look)
  - Smooth animated UI elements

- **Player & World Mods**
  - Speed & jump multipliers
  - Fly / noclip modes
  - Invisible / ghost mode
  - Teleport to players / waypoints

- **ESP & Visuals**
  - Player ESP (boxes, tracers, skeletons)
  - Item / object highlighting
  - Wallhack / see-through walls

- **Fun & Trolling**
  - Voice chat soundboard / spam
  - Fake lag / rubberbanding
  - Size changer (grow/shrink players)
  - Particle spam & screen effects

- **Safety & Anti-Detection**
  - Toggleable features to stay discreet
  - Custom layer masks to avoid detection
  - Quick-disable hotkey

- **UI & Controls**
  - Clean, responsive menu interface
  - Controller-friendly navigation
  - Save/load config presets

…and many more being added regularly!

## 📥 Installation

1. Download the latest release from → [Releases](https://github.com/hatchson/DiabloMenu/releases)
2. Extract the files
3. Run `DiabloMenu.exe` or inject via your preferred mod loader
4. Enjoy the power of Diablo 🔥

> **Note:** This is a client-side mod for Gorilla Tag. Use at your own risk — modding may violate game ToS.
